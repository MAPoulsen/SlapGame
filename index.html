    Welcome to "Hanz and Franz". This is a fun little game I started when I was first learning to code. I created the background using Photoshop and this game has 10 different gifs. Each gif has somewhere between 4-23 images. This game uses HTML5, CSS, and jQuery.

    I felt like this was a unique opportunity to employ some of my own interests and themes within my designs. I decided to use a creepy vintage circus theme. I found many vintage images of side show actors, banners, tents, etc. and used them in this game.

    Enjoy!
